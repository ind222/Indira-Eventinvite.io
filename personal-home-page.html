Assign

This section is about the assignment you have to complete. The assignment requires creating a fully functional webpage using only HTML. You must include a header, a navigation bar, a main section with at least two subsections, an image, and a video. Each section should contain meaningful content and the entire page should have more than one hundred words. Pay close attention to details like semantic tags, accessibility, and structure. Make sure you use correct syntax and properly close all HTML elements. This exercise will help you become more confident in writing clean and correct HTML code, which is fundamental for web development.
